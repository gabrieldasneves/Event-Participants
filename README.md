# Event-Participants

## A simple project developped to review some React Native's foremost concepts.

### Technologies:

- Expo
- React Native
- Typescript

### Revised concepts:

- Render flow
- refresh and reload
- ScrollView vs Flatlist
- Hooks concept

### Run it:

- Donwload the code
- Inside the folder Event-Participants run:
 npm i
- then run:
 expo start
- scan the QR code and have fun!
